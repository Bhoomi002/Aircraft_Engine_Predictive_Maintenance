## 📖 Overview

This project predicts the Remaining Useful Life (RUL) of aircraft engines using a Transformer-GRU deep learning model and NASA's C-MAPSS dataset.

## 🚀 Features

- Predict Remaining Useful Life (RUL)
- Real-time engine health monitoring
- Multi-engine comparison dashboard
- Degradation forecasting
- Maintenance history management
- Engineer and Admin dashboards

## 🛠️ Tech Stack

- Python
- TensorFlow
- Keras
- Streamlit
- Pandas
- NumPy
- Scikit-learn
- SQLite

## 📊 Model Performance

| Metric | Score |
|---------|--------|
| Validation R² | 0.91 |
| Last-Cycle Test R² | 0.89 |

## 📂 Dataset

NASA C-MAPSS FD001 Dataset

Files:
- train_FD001.txt
- test_FD001.txt
- RUL_FD001.txt
