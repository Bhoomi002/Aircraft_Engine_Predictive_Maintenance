# Dataset

This project uses the NASA C-MAPSS FD001 dataset for Remaining Useful Life (RUL) prediction.

Dataset source:
https://www.nasa.gov/content/prognostics-center-of-excellence-data-set-repository

Files used:
- train_FD001.txt
- test_FD001.txt
- RUL_FD001.txt

The dataset is not included in this repository due to size and licensing considerations. Please download it from the official NASA repository.
